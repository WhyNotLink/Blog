## 一、基本语法

### 1.1 程序结构

{% raw %}

```c
#include <stdio.h>
int main() {
    // 主函数，程序入口
    printf("Hello World");
    return 0;
}
```

{% endraw %}

- 必须包含`main`函数，程序从这里开始执行
- 语句以 ** 分号；** 结尾
- 注释：`//`单行、`/* */`多行

### 1.2 标识符与关键字

**标识符就是编程里给「变量、函数、类、数组、结构体、模块」等起的名字**，相当于给程序里的各种东西贴**专属标签**，方便程序识别和调用。

**关键字**就是系统已经占用的名字，有特殊含义，不能随便用。

- 命名规则：字母 / 下划线开头，区分大小写，不能用关键字
- 常用关键字：`int`、`if`、`for`、`return`、`struct`、`void`等

## 二、数据类型

### 2.1 基本数据类型

| 类型      | 说明       | 典型字节 |
| --------- | ---------- | -------- |
| char      | 字符型     | 1        |
| int       | 整型       | 4        |
| float     | 单精度浮点 | 4        |
| double    | 双精度浮点 | 8        |
| long      | 长整型     | 4/8      |
| long long | 超长整型   | 8        |

### 2.2 类型修饰符

- `signed`：有符号（默认）
- `unsigned`：无符号，只存正数
- `short`：短整型

### 2.3 常量与变量

- 变量定义：`类型 变量名 = 值;`
- 常量：`const 类型 名 = 值;` 或 `#define 名 值`

## 三、运算符

### 3.1 算术运算符

| 符号 | 名称 | 示例        | 说明                     |
| ---- | ---- | ----------- | ------------------------ |
| `+`  | 加法 | `a + b`     | 两数相加                 |
| `-`  | 减法 | `a - b`     | 两数相减                 |
| `*`  | 乘法 | `a * b`     | 两数相乘                 |
| `/`  | 除法 | `a / b`     | 两数相除（整数相除取整） |
| `%`  | 取余 | `a % b`     | 求两数相除的余数         |
| `++` | 自增 | `a++`/`++a` | 变量值 + 1               |
| `--` | 自减 | `a--`/`--a` | 变量值 - 1               |

### 3.2 关系运算符

| 符号 | 名称     | 示例     | 结果                 |
| ---- | -------- | -------- | -------------------- |
| `>`  | 大于     | `a > b`  | 成立为 1，不成立为 0 |
| `<`  | 小于     | `a < b`  | 成立为 1，不成立为 0 |
| `>=` | 大于等于 | `a >= b` | 成立为 1，不成立为 0 |
| `<=` | 小于等于 | `a <= b` | 成立为 1，不成立为 0 |
| `==` | 等于     | `a == b` | 成立为 1，不成立为 0 |
| `!=` | 不等于   | `a != b` | 成立为 1，不成立为 0 |

### 3.3 逻辑运算符

| 符号 | 名称   | 规则               | 说明                       |
| ---- | ------ | ------------------ | -------------------------- |
| `&&` | 逻辑与 | 全真为真，一假即假 | 两边条件都成立，结果才为真 |
| \|\| | 逻辑或 | 一真即真，全假为假 | 只要一边成立，结果就为真   |
| `!`  | 逻辑非 | 真变假，假变真     | 对条件结果取反             |

### 3.4 赋值运算符

| 符号 | 作用     | 示例     | 等价于           |
| ---- | -------- | -------- | ---------------- |
| `=`  | 赋值     | `a = 10` | 给变量 a 赋值 10 |
| `+=` | 加赋值   | `a += 5` | `a = a + 5`      |
| `-=` | 减赋值   | `a -= 5` | `a = a - 5`      |
| `*=` | 乘赋值   | `a *= 5` | `a = a * 5`      |
| `/=` | 除赋值   | `a /= 5` | `a = a / 5`      |
| `%=` | 取余赋值 | `a %= 5` | `a = a % 5`      |

### 3.5 位运算符

| 符号 | 名称     | 作用                                 |
| ---- | -------- | ------------------------------------ |
| `&`  | 按位与   | 对应位都为 1，结果为 1               |
| \|   | 按位或   | 对应位有 1，结果为 1                 |
| `^`  | 按位异或 | 对应位不同，结果为 1                 |
| `~`  | 按位取反 | 所有位 0 变 1、1 变 0                |
| `<<` | 左移     | 二进制位左移，高位丢弃，低位补 0     |
| `>>` | 右移     | 二进制位右移，低位丢弃，高位补符号位 |

## 四、流程控制

### 4.1 条件语句

{% raw %}
```c
// if-else
if(条件) {
    语句;
} else if(条件) {
    语句;
} else {
    语句;
}

// switch-case
switch(变量) {
    case 1: 语句; break;
    case 2: 语句; break;
    default: 语句;
}
```

{% endraw %}

### 4.2 循环语句

{% raw %}
```c
// for循环
for(初始化; 条件; 增量) {
    循环体;
}

// while循环
while(条件) {
    循环体;
}

// do-while（至少执行1次）
do {
    循环体;
} while(条件);
```

{% endraw %}

### 4.3 跳转语句

- `break`：跳出循环 /switch
- `continue`：跳过本次循环
- `return`：函数返回
- `goto`：无条件跳转（少用）

## 六、函数

### 6.1 函数定义与调用

{% raw %}

```c
// 定义
返回值类型 函数名(参数) {
    函数体;
    return 值;
}

// 示例
int add(int a, int b) {
    return a + b;
}
```

{% endraw %}

### 6.2 函数分类

- 无参函数：`void func()`
- 有参函数：带形参
- 无返回值：`void`类型

### 6.3 函数声明

- 调用前必须声明 / 定义。这里指的是如果你函数构造体在你这行代码后面，用函数时要提前声明，因为程序是自上而下运行，不声明编译器就找不到你写的函数。

```c
int add(int a, int b); // 声明
```
